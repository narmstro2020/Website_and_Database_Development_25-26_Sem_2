Demo 1: Basic Click Event

The simplest event - clicking a button. We use addEventListener() to listen for clicks.

Click the button to see the result...
JavaScript
// Select the element
const btn = document.querySelector('#demo1-btn');

// Add an event listener
btn.addEventListener('click', () => {
    // This code runs when clicked!
    console.log('Button was clicked!');
});

Demo 2: Multiple Mouse Events

Elements can listen for many event types! Try mouseenter, mouseleave, and click.

Hover & Click Me!
JavaScript

Demo 3: Keyboard Events

The event object contains info about what happened. For keyboards, check event.key and event.code.

JavaScript
const input = document.querySelector('#demo3-input');

// The callback receives an 'event' object!
input.addEventListener('keydown', (event) => {
    console.log(event.key);      // "a", "Enter", "ArrowUp"
    console.log(event.code);     // "KeyA", "Enter", "ArrowUp"
    console.log(event.shiftKey); // true or false
    
    // Check for specific keys
    if (event.key === 'Enter') {
        console.log('User pressed Enter!');
    }
});

Demo 4: Form Submit & preventDefault()

Forms try to refresh the page by default. Use event.preventDefault() to stop that and handle data with JavaScript!

Form data will appear here...
JavaScript
const form = document.querySelector('#demo4-form');

form.addEventListener('submit', (event) => {
    // STOP the page from refreshing!
    event.preventDefault();
    
    // Now handle the form data with JS
    const formData = new FormData(event.target);
    const username = formData.get('username');
    const email = formData.get('email');
    
    console.log('Submitted:', username, email);
});

const list = document.querySelector('#demo5-list');

// ONE listener on the parent catches ALL child clicks!
list.addEventListener('click', (event) => {
    // event.target = the actual element clicked
    // event.currentTarget = the element with the listener (the ul)
    
    // Check: was it a delete button?
    if (event.target.matches('.delete-btn')) {
        // Find the parent <li> and remove it
        const listItem = event.target.closest('li');
        listItem.remove();
    }
});
